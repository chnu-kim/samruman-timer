# 데이터 기반 기능 기획

## 입력

- 영역: `timer`, `project`, `social`, `analytics`, `monetization`, `ux`
- 주제: 자유 텍스트 (예: "시청자 참여 기능", "모바일 경험 개선")
- `all` — 전체 프로젝트 분석 후 기능 제안

## 동작

**반드시 `product-planner` 에이전트를 Agent 도구로 호출하여 위임한다.**

에이전트 프롬프트에 사용자가 지정한 대상(영역/주제)을 전달하고,
에이전트가 반환한 결과(시장 분석 + 경쟁사 비교 + 기능 제안 + 로드맵)를 그대로 출력한다.
기획서 출력 후 구현 여부는 사용자가 결정한다.

## 출력 형식

에이전트가 반환하는 형식을 그대로 사용한다:

```
## 📋 기능 기획서: [주제]

### 시장 환경 요약
### 경쟁사 분석
### 기능 제안 (테이블)
### 상세 기획
### 기능 제거 제안
### 추천 로드맵
```
